// ABOUT

A judge that turns
source into a verdict.

Verwandlung is an open-source, cross-platform online judge built on Spring MVC and a distributed pool of judgers coordinated through ActiveMQ. It compiles, runs, and grades submissions against hidden test cases in milliseconds — the same engine behind every contest and practice problem on this site.

// ARCHITECTURE
Spring MVC
Java backend serving problems, submissions, contests and the REST API.
ActiveMQ
Message broker that distributes judge jobs across the judger pool.
Judgers
Sandboxed runners compiling & executing code under strict time/memory limits.
MySQL
Persistent store for users, problems, test data and verdicts.
Redis
Caches sessions, rate limits and live contest standings.
Vanilla JS
Lightweight front-end — no heavyweight SPA framework.
